# C++ CLI Experiments & Utilities ⚡

Welcome to my personal collection of native C++ command-line applications and programming exercises. This repository serves as a practical playground for implementing core data structures, automation logic, and console-based systems.

## 🚀 What's Inside?

* **Academic & Management Tools:** Features robust console systems like `studentGrades.cpp` to handle dynamic records and automate academic tracking.
* **Interactive Logic & Games:** Custom behavioral scripts and text-driven logic pathways, such as `theHeartBrokenIChoose.cpp`.
* **Inventory & Data Systems:** Implementations like `Inventory.cpp` and `playerInventory` designed to manage structured data efficiently in memory.
* **Core Fundamentals:** Foundational programming concepts and array manipulation utilities (`array.cpp`, `demo.cpp`).

## 🛠️ Getting Started

### Prerequisites
To compile and run these utilities, make sure you have a C++ compiler installed (such as `g++` or `clang`).

### Compilation
You can compile any individual tool directly from your terminal. For example, to run the student grades system:

```bash
g++ -std=c++17 studentGrades.cpp -o studentGrades
./studentGrades
